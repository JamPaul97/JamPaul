Recipe Editor

About

Scrap Mechanic Survival introduced crafting recipes. Those recipes can be editied by editing the json files that contains the recipes themself.
Recipe Editor making this as easy as to drag and drop the items to change the inpup/output, the crafting time and the ammount of the output..
Recipe Editor use the game file's, that means that any mod items will be visable in the editor(WIP).
